# Enygma Project

Página dedicada ao artista independente Enygma, desenvolvido como projeto de estudo em Front-End.

## Estado atual do projeto:

 EM DESENVOLVIMENTO - Versão 0.2

## Objetivo

O intuito deste projeto é praticar conceitos fundamentais do desenvolvimento web, como:
 - Estruturação em HTML;
 - Estilização com CSS;
 - Uso de Flexbox;
 - Organização de arquivos.

## Tecnologias utilizadas até o momento
 - HTML5
 - CSS3
